ALL ABOUT HTML TAGS



  1. DIV tag: is used to group elements together or define sections of content.
  2. Headings (h1,h2,h3,h4,h5,h6):These are used to define heading of the content with ranging from h1 to h6 having different size.
  3. P tag: is used to define a paragraph.
  4. HR Tag:It is used to insert a horizontal line.
  5. BR Tag:It is used to break a line and start with a new line.
  6. Span Tag:It allows us to markup and highlight specific portion of text within the document.
  7. A Tag:It is defined as Anchor tag and is used to create a link for going to other webpages. It creates the link using href attribute which is defined within the tag itself.
  8. UL Tag:It is used to create an unordered list defined using LI Tag.
  9. OL Tag:It is used to create an ordered list defined using LI Tag.
  10. DL Tag:It is defined as Definition list. It represents a list of Definition terms defined in DT Tag and their corresponding descriptions defined in DD Tag.
  11. TABLE Tag:It is used to create a table.
  12. TR Tag:It is used to define Table row.
  13. TD Tag:It is used to define Table data.
  14. B or Strong Tag:It is used to make a specific text bold.
  15. I Tag:It is used to make the text style italic.
  16. EM Tag:It is used to indicate emphasis.
  17. MARK Tag:It is used to make a specific text highlighted with a particular color.
  18. SMALL Tag:It is used to make a specific text look small.
  19. DEL or S Tag:It is used to overwrite a strikethrough line over a exisiting text to indicate removed text.
  20. INS or U Tag:It is used to underline a particular text.
  21. SUB Tag:It is used to render text as subscripted positioning it slightly below the normal text baseline.
  22. SUP Tag:It is used to render text as superscripted positioning it slightly above the normal text baseline.
  23. BIG Tag:It is used to make the text look big as compare to normal text.
